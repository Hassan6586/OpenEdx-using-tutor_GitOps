================================================================================
                         QUICK DEPLOYMENT
================================================================================

Prerequisites:
- AWS Account with credentials configured
- Terraform >= 1.0
- kubectl >= 1.25
- Helm >= 3.10
- Domain name (e.g., openedx.example.com)

Deploy in 3 steps:

1. Navigate to project:
   $ cd openEdx-eks

2. Configure environment:
   $ cp terraform/terraform.tfvars.example terraform/terraform.tfvars
   $ nano terraform/terraform.tfvars  # Edit with your values

3. Deploy everything:
   $ make setup
   
   ⏳ Takes approximately 15-20 minutes
   
   The 'make setup' command runs:
   - terraform init
   - terraform validate
   - terraform plan
   - terraform apply
   - kubeconfig setup
   - OpenEdX deployment
   - final verification

Verify deployment:
   $ make status

Access OpenEdX:
   1. Get LoadBalancer URL: kubectl get svc -n ingress-nginx
   2. Configure DNS to point to LoadBalancer hostname
   3. Visit https://openedx.example.com
